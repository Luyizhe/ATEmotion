# ATEmotion
这是一个用作情感识别的python库。

支持音频/文本单模态，并且也支持两个模态的融合。

支持以单个句子为单位训练模型，并且预测单个句子的情感。也支持以对话为单位，并且预测对话中每个句子的情感。

可以直接通过```pip install ATEmotion```
安装本库，或者通过
```git clone https://github.com/Luyizhe/CN_Emotion.git```将代码下载到本地使用。
